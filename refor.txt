In this chapter, a novel hybrid methodology for wind speed forecasting (WSF) by combining improved complete ensemble empirical mode decomposition with adaptive noise (ICEEMDAN) and a memory-efficient Reformer (REF) model is proposed. This approach involves decomposing the input data intoo the noiseless data using the from the ICEEMDAN algorithm. The resulting sub-series are then processed through a Reformer model to forecast wind speed. The effectiveness and development of the proposed ICEEMDAN-REF model is validated using wind speed data from two distinct locations, Block Island and the Gulf Coast. Additionally, the proposed model is evaluated across four time horizons: 5-min, 15-min, 30-min, and 1-hour. 


\section{Introduction}
The self-attention mechanism of the Transformer model, introduced in Chapter 3, has been employed to acquire knowledge of complex patterns and dynamics in wind speed data. The Transformer model has shown impressive ability to capture long-range dependencies, outperforming other comparative models. However, it faces limitations such as high spatial and temporal complexities, restricted input-output sequences, and the high computational cost of training, especially for long sequence processing in WSF tasks. These challenges highlight a critical opportunity to enhance the Transformer model for more optimal WSF performance. As a result, in this chapter, a novel and hybrid ICEEMDAN-REF model for WSF is proposed to overcome the limitations identified in the ICEEMDAN-TRA model discussed in the Chapter 3. The following are the key contributions of this research work:

\begin{itemize}
\item {In this chapter a novel hybrid methodology known as ICEEMDAN-REF is proposed for accurate WSF}
\item {To address the challenges of the CEEMDAN algorithm, including mode mixing, frequency aliasing, and residual noise, the ICEEMDAN algorithm is used to effectively denoise the input wind speed data.}
\item{To address the challenges encountered with the Transformer model, including high spatial time complexities, restricted input-output sequences, and training time, a robust REF model is developed for effective WSF. The REF model is significantly more memory-efficient and exhibits faster processing times when handling long sequences in the WSF. This efficiency is achieved through the incorporation of LSH and reversible-residual layers.}
\item {To validate the effectiveness of the proposed approach, the study conducted at two different wind farms using four different time horizons: 5-min, 15-min, 30-min, and 1-hour ahead WSF.}
\end{itemize}

This chapter is organized as follows: Section 2 provides a demonstration of the operational principles of the hybrid ICEEMDAN-REF methodology proposed. Section 3 presents the experimental results using two case studies and evaluation conducted. The final section summarizes the key findings and implications of this study.

\subsection{Proposed REF model for the WSF}
\subsubsection{Issues with Transformer model} 
RNN variations like LSTM and GRU have been used for sequential forecasting in the literature. These models are computationally expensive and have a restricted receptive field, which can impair their effectiveness in WSF. In chapter 3 the Transformer technique is proposed to address these restrictions. The Transformer model has been developed by removing repetition and introducing multi-head self-attention \cite{ref_30}. Parallel processing allowed the Transformer to handle bigger sequences, creating sophisticated models with less training time. Though Transformer models excel at handling longer sequences, they have the following limitations:  
\begin{enumerate}
\item Attention computation: Attention computations upon L-length sequences is $O(L^2)$ (in both time as well as in memory)
\item Large number of layers: To facilitate back-propagation, the activations within all layers must be retained. As a result, a model comprising N-layers requires N times much more memory when compared with a single 	layer model.
\item Depth of feed forward layers: The intermediary feed-forward layers tend to have greater depth when compared to the depth of attention activations. This comprises a significant amount of memory use.
\end{enumerate}

To address Transformer model limitations, a memory-efficient REF model has been developed for WSF. The REF model incorporates three key techniques such as: 1) LSH Attention, 2) Reversible layers, 3) Chunking. A comprehensive explanation of these techniques is provided in the subsequent subsections:
\paragraph{LSH attention}
In DL, attention mechanisms enable neural networks to focus on particular elements at each time step. In the Transformer model, three key types of attention mechanisms are utilized: encoder self-attention, masked decoder self-attention, and encoder-decoder attention. The conventional approach for attention in the Transformer model uses scaled dot-product, which can be defined mathematically as follows:
\begin{equation}
\label{ref_e6}
Attention(X,Y,Z)=softmax (\frac{XY^T}{\sqrt{d_Y}})Z
\end{equation}

\textit{Where, X represents L Queries with dimension d, Y represents L Keys with dimension d, Z represents L Values with dimension d, L represents sequence length and d represents attention depth. }

From the preceding equation, the computational as well as memory cost of performing multiplication $XY^T$, where the shape of the matrix is $[L, L]$, can be observed to be in $O(L^2)$ which indicates that memory is the primary bottleneck. The computation and storage of the entire matrix \( XY^T \) are unnecessary since the focus is on \( \text{softmax}(XY^T) \). This computation is primarily influenced by the most significant elements in a matrix that is usually sparse. Therefore, to optimize the search process, it is essential to focus solely on the keys Y that are in close proximity to the query X. The attention mechanism has been observed to identify the closest neighbor keys of a query, although with inadequate efficiency. Consequently, a nearest neighbor's search is necessary. The main development in the REF is the substitution of a dot-product attention mechanism by the LSH. This substitution has resulted in a notable decrease in computational complexity, from $O(L^2)$  to $O(L logL)$.

The fundamental concept underlying LSH attention can be described as follows. The attention mechanism is modified by restricting the computation of attention to specific vectors in the X and Y matrices, instead of to all vectors, as shown in equation \ref{ref_e6} of the standard attention formula. To enhance the probability of dissimilar items being allocated to different buckets, repeated the aforementioned process several times.

\begin{itemize}
\item Obtain the X and Y matrices of LSH hashes.
\item The standard attention is computed only with x and y vectors that belong to the same hash buckets.    
\end{itemize}

\paragraph{Reversible residual Network (RevNet)}
Looking closely at the Transformer model's encoding and decoding blocks, the attention and feedforward layers are grouped into residual blocks \cite{ref_50}. ResNets, introduced in \cite{ref_45}, are a strong component used in NNs to address the vanishing gradient problem in DNNs. Because ResNets must store layer activations in memory for gradient calculation during backpropagation, memory use is a major issue. Results show a clear relationship between neural network size and memory resources needed to hold parameters. RevNet uses residual modules to tackle this problem. Revnet's architecture permits perfect reconstruction of each layer's activations from its follow-up activations. This feature permits backpropagation without memory activation storage. In Figure \ref{Rev}, residual and reversible modules are shown. The image shows how to calculate the block's inputs ($I_1$, $I_2$) from its outputs ($O_1$, $O_2$). The Transformer's second issue is N-layer Transformer network storage, which may be crucial for big N values. In the REF model, the RevNet idea integrates the attention and feed-forward layers. The attention layer is marked by A in Fig. \ref{Rev}, while the feed-forward layer is denoted by F. Reversible residual layers allow activation storage once during training instead of N times.

\begin{figure}
\centering
\includegraphics[width=5 in]{Figures/RevNETT.eps}
\caption{ResNet Modules (left) and RevNet Modules (right)}
\label{Rev}
\end{figure}

\begin{align}
\label{ref_e7}
    O_1 = I_1 + A(I_2) \nonumber \\
    O_2 = I_2 + F(O_1)
\end{align}

\paragraph{Chunking}
The REF model's third efficacy improvement involves high-dimensional intermediary vectors in feedforward layers. Reversibility does not solve memory use because thicker layers require more memory. Feedforward layer calculations are independent by sequence position, therefore they can be chunked. This allows chunking of forward, backward, and reverse calculations. Traditional processing of this layer uses batch-based parallel execution of operations across all positions. Processing one chunk at a time reduces memory use.


\begin{equation}
\label{ref_e8}
O_2 \textbf{=} [O_2^{(1)};\dots;O_2^{(c)}] \textbf{=} [I_2^{(1)}+FeedForward(O_1^{(1)} );\dots;I_2^{(c)}+FeedForward(O_1^{(c)})]     
\end{equation}

\subsubsection{Architecture of the REF model}
In this study, the REF model as shown in the Fig. \ref{REF}, is used for WSF. REF composed of two core elements: encoder and decoder. \\
\textbf{Encoder:}\\
In Encoder, historical wind speed data is the input, undergoing a sequence of transformations. It first passes through a stack of fully connected layers, extracting significant features from the wind speed dataset. Subsequently, positional encoding is applied to retain important temporal information, ensuring the model comprehends the sequential order of data. Efficient LSH attention is then applied which divides the input sequence into manageable chunks or segments, allowing the model to concentrate on the most relevant information within these segments. By efficiently capturing dependencies within these local segments, LSH attention significantly reduces the computational burden of conventional self-attention mechanisms while preserving the model's capacity to recognize temporal patterns in the wind speed data. The resulting output from the LSH attention mechanism is then augmented with the original input via residual connections, and layer normalization ensures that the model maintains stable and effective training throughout the learning process. Then, this output is passed through the feedforward chunking process, enabling efficient handling of long sequence. Processing one chunk at a time reduces the memory use through the Eq. \ref{ref_e8}. This result is passed through the RevNet module which integrates the attention and feedforward layers effectively. The proposed REF model is designed to handle long sequences more efficiently by using a reversible network architecture and utilizing the LSH technique. The mathematical formulation of RevNet is represented in the Eq. \ref{ref_e7}. This enables the model to process longer sequences without significantly increasing computational requirements and module helps in the storage of activation once during the training instead of N number of times, where N is the no. of encoder layers. Importantly, this encoder architecture comprises multiple identical layers, each repeating the sequence of operations, ultimately yielding a sequence of encoded representations. These operations collectively generate highly informative encoded representations, combining feature extraction, efficient dependency capture, and information preservation to generate input to the decoder network. \\
\textbf{Decoder:}\\
In the REF model's Decoder, the operations begins with the last recorded wind speed measurement subjected to a fully connected layer for feature extraction, then positional encoding to maintain temporal information. The residual layer is present to ensure the training stability. The LSH attention is then applied, taking inputs from the preceding addition and normalization step and the Encoder's output, effectively capturing dependencies within localized segments. Subsequent addition and normalization ensure consistent activations for the stable training. The data is further processed through chunking for memory optimization and refinement. The RevNet module integrates attention and feedforward layers efficiently, storing activations once during training. Finally, a linear dense layer generates wind speed predictions, aligning with the desired forecasting horizon as shown in the Figure. \ref{ref_proposed}.

\section{Summary}
In this chapter, a novel and hybrid ICEEMDAN-REF model for WSF is proposed by integrating ICEEMDAN and REF model. The ICEEMDAN algorithm is utilized for denoising the available wind speed data, and then the denoised data is passed for the prediction using REF model. The analytical results demonstrated the proposed ICEEMDAN-REF model produced the reliable WSF results. Two experimental studies have been performed to assess the efficacy of the proposed ICEEMDAN-REF model for WSF at 5-min, 15-min, 30-min, and 1-hour intervals. This experiment evaluates the performance of two groups of WSF models and compares their results with the proposed ICEEMDAN-REF model. These results reveal that the proposed model delivers superior results, achieving a statistically significant improvement over the other models. Hence, the proposed ICEEMDAN-REF model possesses the capability to aid power system operators to achieve the optimum balance among demand and generation within the modern power system. The hybrid prediction model provided exhibits versatility and suitability for various forecasting application scenarios, thereby enhancing efficiency.
